To download the necessary files for this folder, visit:

https://osf.io/k6vx8/files/osfstorage

Instructions:
1. Click on the “Files” tab at the top.
2. Click on the three dots next to the "priors" folder.
3. Select “Download” to download the entire folder as a ZIP file.
4. Extract the contents into this 'data_OSF' folder in your local project.
5. Make sure the folder structure remains unchanged.
6. The scripts in this project expect the data to be located here.
